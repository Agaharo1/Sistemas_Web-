Planificación

Introducción:

Con el fin de organizar la tareas de manera ordenada y precisa se presenta la siguiente planificación con los siguientes objetivos:

  • Crear una página web con funcionalidades para compra-venta, chat entre usuarios, y gestión de productos.
  • Asegurar que los productos sean verificados para garantizar la calidad y autenticidad de los mismos.
  • Implementar un sistema de pujas y promoción de productos mediante anuncios pagos.
  • Desarrollar un diseño atractivo, intuitivo y fácil de usar para los usuarios.
